## Leh We Fen Wok(overview)
Leh We Fen Wok is a responsive landing page created to help young people in Sierra Leone discover jobs, internships, scholarships, and training opportunities through one digital platform.
The website was developed using HTML and CSS and is designed to work on desktop, tablet, and mobile devices.

## Features
- Responsive design
- Modern landing page
- Job and training information
- Contact form
- Mobile-friendly layout

  
## Technologies Used
- HTML
- CSS
- SDG Alignment

This project supports SDG 8  Decent Work and Economic Growth.
